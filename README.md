# FundFusion

**FundFusion** is a modern fundraising web application built to support various causes and individuals in Bangladesh. The platform enables users to create, manage, and donate to campaigns, with a focus on user-friendly functionality and real-time updates.

## Features

- **Campaign Browsing:** Non-registered users can browse live campaigns, view details, and make donations seamlessly without needing to sign up.
- **Campaign Management:** Registered users can create, manage, and track their fundraising campaigns with ease.
- **Real-time Updates:** Display live fundraising progress with dynamic updates on campaign amounts (Raised/Target), without requiring page refreshes.
- **Interactive UI:** A smooth and responsive UI designed with Fluent UI for effortless navigation between tabs, coupled with a glassy, wavy aesthetic for a modern feel.
- **Admin Control:** Full admin control over user management, campaigns, payouts, and platform statistics.
- **Responsive Design:** Designed for both desktop and mobile users for easy access to campaigns on the go.

## Key Functionalities

1. **Live Fund Progress:** Each campaign displays a dynamically updated progress bar, showing real-time raised and target amounts in a stylish, interactive box.
2. **Bold Campaign Display:** The campaign titles are displayed using bold 'Sergio UI' text with rounded corner campaign cards.
3. **Campaign Dashboard:** Users can track and manage their donations and supported campaigns from a personalized dashboard.
4. **Recurrence Options:** Registered users can set up recurring donations for their favorite causes.
5. **Informative Pages:** Additional sections like "About Us" and "FAQ" to provide essential information about the platform.
